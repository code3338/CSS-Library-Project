GarageCSS is a simple JavaScript CSS library with a variety of different features. Features include headings,links,buttons, nav bars, grids,and forms.

Getting Started:

  1. Select the download button at the top of the page, and download the GarageCSS file.
  2. In your HTML file, link the css file at the top of your HTML file.
  3. Also add <link rel="stylesheet" href="w3Schools.css"> to the top of your HTML file.
  4. You are now ready to begin using GarageCSS.

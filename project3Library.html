GarageCSS is a simple CSS library with a variety of different features. Features include headings, links, buttons, nav bars, grids, forms, and more.

Getting Started:

  1. Select the download button at the top of this page, and then clicking the "GarageCSS version 1" link to download the GarageCSS .css file.
  2. In your HTML file, link the garageCSS.css file to the top of your HTML file.
  3. If you choose to use the radio button styles, download the w3Schools.css file by selecting the download button at the top of this page, and then clicking the "w3Schools" link to download the w3Schools .css file.
    After downloading the file,add <link rel="stylesheet" href="w3Schools.css"> to the top of your HTML file.
  4. If you choose to use the toggle button download garageCSS.js file by selecting the download button at the top of this page, and then clicking the garageCSS.js link to download the GarageCSS JavaScript file. After downloading the file, add <link rel="stylesheet" href="garageCSS.js"> to the top of your HTML file.
  5. You are now ready to begin using GarageCSS.

GarageCSS

  • Download
  • Documentation

GarageCSS is a simple JavaScript CSS library with a variety of different features. Features include headings,links,buttons, nav bars, grids,and forms.

Getting Started:

  1. Select the download button at the top of the page, and download the css file.
  2. In your HTML file, link the css file in between the tags.
  3. You are now ready to begin using GarageCSS.

Use the following grid styles. Simply add the class "gridDiv" to the column container, and the div's(or colunns) within the container add the class "col" and a second class of "col_1 to col_5" to create different styles of grids.
